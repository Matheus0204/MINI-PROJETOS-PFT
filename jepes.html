Mini-Projeto JEPs: Verificação de Aptidão de Atletas

Este projeto simula a lógica para determinar se um atleta pode competir nos Jogos Escolares de Pernambuco.

Para ver o resultado, abra o Console do Desenvolvedor do seu navegador:

  1. Pressione F12 (na maioria dos navegadores).
  2. Vá para a aba Console.

Você pode editar os valores das variáveis no arquivo jeps.js para testar diferentes cenários de atletas!
